# Digitalised Height and Weight Measurement System

## 📌 Overview
This project is based on Arduino and focuses on measuring a person’s height and weight digitally with improved accuracy and minimal human effort. It integrates sensors and display modules to provide real-time readings, making the process efficient and user-friendly.

## ⚙️ Components Used
- Arduino board  
- Ultrasonic Sensor (for height measurement)  
- Load Cell (for weight measurement)  
- HX711 Amplifier Module  
- LCD / OLED Display  
- Resistors  
- Breadboard & connecting wires  

## 💡 Features
- Accurate digital measurement of height and weight  
- Real-time display of readings  
- Reduces manual errors  
- Compact and easy-to-use system  

## 📂 Files
- `HEIGHT_SENSOR.ino` → Measures height using ultrasonic sensor  
- `WEIGHT_SENSOR.ino` → Measures weight using load cell  
- `DISPLAY.ino` → Displays output on screen  

## 🚀 Working Principle
- Ultrasonic sensor measures distance from head to sensor → calculates height  
- Load cell detects weight and sends signal to HX711 amplifier  
- Arduino processes both inputs  
- Final height and weight are displayed on screen
